To use this custom control in a XAML file in another project, complete the following steps:
1) Add a reference to this project.

2) Add the following line to the root element of the XAML file where you wish to use this
   control:
   xmlns:MyNamespace="clr-namespace:JCsTools.JCQ.Controls;assembly=JCsTools.JCQ.Controls"

3) Use the control in the XAML file:
   <MyNamespace:CustomControl1/>
